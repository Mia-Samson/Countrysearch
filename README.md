## ✨ Features

- ⚡ Real-time search filtering
- 🌍 Country dropdown list
- ⌨️ Type-ahead suggestions
- 🎯 Case-insensitive search
- 🖱️ Click to select option
- ❌ No match handling
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
